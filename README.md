### 🎯 What is Token Minimizer?

Token Minimizer (aka "抠门 Skill") is an LLM token optimization experiment. It achieves **20x compression with <2% performance loss** using research-backed methods from top AI labs. Just for fun, and for saving your hard-earned money.

> "Being stingy with tokens is the highest form of engineering." — Some engineer, probably

### 🔬 Research Backing

| Method | Institution | Compression | Performance Impact |
|--------|-------------|-------------|--------------------|
| LLMLingua | Microsoft Research | 20x | -2% |
| ProCut | EMNLP 2025 | 8x | -3% |
| 500xCompressor | Stanford | 6-480x | -5% |
| Granularity Control | Amazon Science | 4x | **+23% GAIN** |

### 🚀 Key Features

- ✅ **Differential Compression**: Critical instructions (0-10%), chitchat (70-90%)
- ✅ **Attribution Pruning**: Remove tokens with <5% output contribution
- ✅ **KV Cache Optimization**: 50-90% cost reduction via prefix reuse
- ✅ **Tool Batching**: Minimize round-trips, maximize information density
- ✅ **Performance Guards**: Never mutilate core capabilities
- ✅ **Bilingual**: Native English & Chinese support
